# Data Insights Demo App

A React application demonstrating the capabilities of the Insight SDK for building data visualizations and insights.

## Features

- **Trend Insight**: Visualize single metrics over time using line charts
- **Contributor Insight**: Show how different dimension values contribute to a metric over time using stacked bar charts
- Built with React, TypeScript, and Vite for fast development
- Styled with Tailwind CSS for modern UI

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Insight SDK
- Recharts

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173` to view the app.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
